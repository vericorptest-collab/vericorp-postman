Ready-to-use Postman collection for the [VeriCorp API](https://rapidapi.com/vericorptestcollab/api/vericorp).

## Quick Start

1. [Import the collection](VeriCorp%20API.postman_collection.json) into Postman
2. [Import the environment](VeriCorp%20API.postman_environment.json)
3. Set your `api_key` in the environment variables
4. Start making requests

## What's Included

| Folder | Requests |
|--------|----------|
| Company Lookup | PT, DK, NO, FI, EE, PL, CZ, UK, DE (9 countries) |
| VAT Validation | Valid + invalid examples |
| Batch Lookup | 3-company batch request |
| Utilities | Health check, supported countries |

Every request includes **test scripts** that validate response status and schema.
